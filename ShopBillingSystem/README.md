 # Shopping Billing System

A desktop-based billing application developed using Java Swing, JDBC, and MySQL.

## Features

- User Login System
- Product Management
- Billing Generation
- Sales Management
- Database Connectivity
- Dashboard Interface

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- VS Code
